# Laftel Sub

Personal subscription tracker — Know what you're paying for.

## Project Info

- **Marketing-ready:** No
- **Code review required:** No

## Architecture

- **Backend**: Express.js REST API with MongoDB
- **Frontend**: React SPA

## Development Setup

### Backend
```
cd backend
npm install
cp ../.example.env .env   # fill in values
npm run dev
```

### Frontend
```
cd frontend
npm install
npm start
```

## Running Tests

```
cd backend && npm test
```

## Environment Variables

See `.example.env` for required variables.
